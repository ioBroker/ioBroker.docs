# IoBroker 月度回顾 – 2026 年 5 月/6 月
2026 年夏季，ioBroker 生态系统可谓热闹非凡！自上次更新以来，我们已在产品库中新增了 15 个适配器。

从 MSpa 按摩浴缸和 AL-KO 机器人割草机，到用于墙板的 Home Assistant 模拟器，应有尽有。

其中，**能源和太阳能**领域尤为突出（例如 HomeWizard、FoxESS、SOLECTRUS、Linky 和 Tibber 小部件），但**位置追踪、监控和照明**领域也占有一席之地。

目前，几乎所有的底层功能都围绕着两个主要主题展开：**管理员**获得了一个中央的**凭证管理器**和一个集成的**AI助手**，**JavaScript适配器**进一步扩展了其AI功能，而**设备适配器**在**v3.0**版本中取得了巨大的飞跃。

## 核心更新
除了新的适配器之外，ioBroker 的核心组件也进行了重要的更新。

### 管理员 7.8.x → 7.9.x
在此期间，管理适配器是最活跃的核心组件。其中两项新功能尤为突出：

- **中央凭证管理器** (v7.9.0) – 访问数据（API 密钥、令牌、云登录）现在可以在一个中心位置进行管理，并可多次重复使用，而不是分别存储在每个实例中。
- **后台管理面板中的 AI 助手**（v7.9.1，默认禁用）– 现在可以直接在后台管理面板中使用 AI 助手。用户可以在助手对话框中方便地创建 API 密钥（v7.9.11），不想使用自己密钥的用户可以通过外部 MCP 客户端运行助手（v7.9.10）。
- **在特定主机上安装软件包** (v7.9.0) – 在多主机设置中，现在可以选择性地将软件包安装到单个主机上。
- **小改进** – 按创建日期对适配器进行排序 (v7.9.4)、实例概览中的端口列可排序 (v7.9.12)、配置管理器中的附加筛选器 (v7.9.6) 以及各种错误修复。

🔗 [GitHub 上的管理员发布](https://github.com/ioBroker/ioBroker.admin/releases)

### Js-controller 7.2.2 “Milla”
js-controller 已发布多个版本——目前，版本 **7.2.2**（2026 年 6 月 16 日）可在 `stable` 代码库中找到。**重要提示：** 7.1.x、7.2.0 和 7.2.1 版本的用户应**尽快**更新。旧版本目前会生成大量集中式错误日志，给 ioBroker 服务器带来不必要的负担。7.2.2 版本解决了此问题。

此外，自 7.1.1 版本起，除了 Docker 之外，**Podman** 和 **CRI-O** 也被认定为容器运行时。

🔗 [GitHub 上的 js-controller 更新日志](https://github.com/ioBroker/ioBroker.js-controller/blob/master/CHANGELOG.md)

### Vis-2 2.14.x
可视化平台vis-2主要进行了内部改进：

- **修订后的构建流程** (v2.14.0)
- 用户自定义样式现在可以正确应用于 `tplValueInput` (v2.14.3)

🔗 [vis-2 在 GitHub 上的发布](https://github.com/ioBroker/ioBroker.vis-2/releases)

## 重要适配器更新
在此期间，两个中央适配器也取得了长足进步。

### IoBroker.javascript v9.2.x – v9.3.1
JavaScript 适配器进一步推进了其 AI 集成，并且也变得更加完善：

- 配置中的**凭据管理器支持** (v9.3.0)
- **运行单向脚本而不保存** (v9.3.1)
- **通过鼠标悬停在编辑器中直接显示状态值** (v9.2.3)
Blockly (v9.2.2) 中新增了 **“状态是否存在”代码块**
- 对所有语言的 AI 聊天模式（代理模式与代码模式）进行更精确的描述
- 进行了多项性能优化和错误修复（包括脚本停止时订阅泄漏的问题）

🔗 [适配器的 GitHub 页面](https://github.com/ioBroker/ioBroker.javascript)

### IoBroker.devices v2.1.x → v3.0.2
设备适配器迎来重大里程碑，版本升级至 3.0。

为庆祝这一里程碑，我们发布了一段全新视频，逐步演示了自动可视化功能，包括切换设备视图和 GUI 视图、使用颜色和背景图像进行房间设计，以及通过二维码将设置传输到 ioBroker VisuApp。

[<img src="https://img.youtube.com/vi/C-e8j8NHdi0/hqdefault.jpg" width="600px" />](https://youtu.be/C-e8j8NHdi0)

最重要的几项新功能一览：

- **自动可视化**功能进一步扩展；只需点击几下鼠标，即可从配置的设备创建完整的用户界面。
- 现在支持**小部件图标**（v3.0.2）
- **类别消耗值** (v3.0.1) 和显示组件的**小数位数** (v2.1.4)
- 支持 **open-meteo-weather** 适配器 (v3.0.0)
- 显著提升了 Safari 浏览器中设备列表的显示流畅度 (v3.0.1)

🔗 [适配器的 GitHub 页面](https://github.com/ioBroker/ioBroker.devices)
