# IoBroker 月度回顾 – 2026 年 3 月/4 月
春天为 ioBroker 生态系统带来了一股清新的气息！2026 年 3 月和 4 月，**13 个全新适配器**被添加到 `latest` 版本库中，涵盖了从地暖、极光预报到智能家居飞行模拟器硬件等各种应用。

**nmea** 适配器也已升级至 `stable` 版本。

不仅如此：**JavaScript** 适配器新增了 AI 聊天面板，**Devices** 适配器采用了全新的可视化界面，**Admin** 适配器通过设备管理器和新的 JsonConfig 功能得到了改进，而 **xterm** 适配器现在拥有了真正的终端体验。

## 核心更新
除了新的适配器之外，ioBroker 的核心组件也进行了重要的更新。

### Js-controller 7.1.1 “Milla”（2026年4月6日）
js控制器已发布新的小版本：

- 增强了容器支持：除了 Docker 之外，现在还支持 Podman 和 CRI-O。这对于依赖其他容器运行时的用户来说非常重要。
- 测试框架现代化（内部，与用户无关）

🔗 [GitHub 上的 js-controller 更新日志](https://github.com/ioBroker/ioBroker.js-controller/blob/master/CHANGELOG.md)

### 管理员 7.8.x – 三月份的改进
管理适配器在 2026 年 3 月进行了多次更新，显著改善了与 ioBroker 的日常协作：

- **设备管理器已修订** (v7.8.17–v7.8.20) – 管理后台中的设备管理功能已得到大幅改进，现在提供了更清晰的界面。
- **Spotify 的 OAuth2 流程** (v7.8.9) – 在 JsonConfig 中，现在可以直接执行 Spotify 的 OAuth2 流程，而无需外部工具。
- **通过文件上传适配器** (v7.8.7/v7.8.8) – 新增了直接从本地文件安装适配器的方法。对开发人员和测试人员非常有用。

🔗 [GitHub 上的管理员发布](https://github.com/ioBroker/ioBroker.admin/releases)

### Vis-2 2.13.x
可视化平台vis-2修复了一些小错误：

- 修正了模式中的 **元音变音处理**
- 修复了通过控制界面传输 JSON 命令的问题

🔗 [vis-2 在 GitHub 上的发布](https://github.com/ioBroker/ioBroker.vis-2/releases)

## 重要适配器更新
除了核心组件外，许多流行的适配器也得到了重要的更新。

### IoBroker.javascript v9.2.0（2026年4月11日 - 测试版）
这段时间以来最大的更新可能是：

- **支持多供应商和智能申请的**AI聊天面板**
- 支持 **Ollama、LM Studio、Google Gemini、DeepSeek、OpenRouter**（版本 9.1.1、19.03）。
- 可配置的 API 基本 URL、动态模型检索、“测试 API”按钮
- Blockly 编辑器中的 **OID 显示模式**（名称、名称路径、州 ID、完整 ID）
- 检查脚本是否已被其他用户/窗口修改

🔗 [适配器的 GitHub 页面](https://github.com/ioBroker/ioBroker.javascript)

### IoBroker.devices v2.0.x（测试版）
设备适配器即将迎来重大升级！如果您还没看过我们关于别名和类别管理的视频，请点击此处观看：

[![视频：带有别名和类别的稳定结构](de/blog/images/2026_04_13_Aliase_Kategorien_YouTube.png)](https://youtu.be/0Aecm5YAk7M)

在学习如何轻松创建设备和别名之后，下一步自然是**自动可视化**。

从**v2.0.3**版本开始，只需点击几下即可可视化适配器中创建的设备，无需手动组装任何组件。适配器会根据现有的设备定义自动生成清晰易用的界面。

无论是灯具、恒温器、百叶窗还是传感器，都会直接提供相应的组件。

关于这项新可视化功能的视频教程已经在制作中。

无需 API 密钥
- 非常适合自动化：例如，当极有可能看到北极光时，激活相机或发送通知。

**类别：** 协议 / 海洋 **开发商：** Bluefox

NMEA 适配器用于连接 NMEA 2000 游艇总线和 ioBroker，现已更新至 `stable` 版本。

这使得船舶的导航数据、环境传感器和自动驾驶系统能够直接集成到 ioBroker 中。

支持的硬件：
- Actisense NGT-1 和 NGX1-USB（USB 转换器）
- 带 PiCAN-M CAN 总线扩展板的树莓派
- Yacht Devices YDWG-02/03（WiFi网关）
- Yacht Devices YDEN-02/03（以太网网关）

#### 支持的数据：
- GPS坐标、温度、湿度、气压
- 自动驾驶仪支持（Raymarine）
- **AIS** – 自动船舶检测
- 数据模拟可用于测试
双向通信
