# IoBroker.hp-ilo ===================
# HP ILO-Adapter für ioBroker
Liest Temperaturdaten von Hewlett-Packard-Servern mit Integrated Lights-Out (ILO) in [ioBroker](https://www.iobroker.net) ein.

Unterstützung von ILO-Schnittstellen ab V2.
