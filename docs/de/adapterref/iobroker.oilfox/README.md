# IoBroker.oilfox
## Beschreibung
Dieser Adapter ermöglicht das Abrufen von Daten vom Oilfox-Sensor über die Oilfox-Website.

## Die Einstellungen
* Geben Sie Ihre E-Mail-Adresse und Ihr Passwort auf der Konfigurationsseite an
