## Hagelschutz VKF-Adapter für ioBroker
Adapter für hagelschutz-einfach-automatisch.ch der kantonalen Gebäudeversicherungen.

> [!WICHTIG] > Dieser Adapter funktioniert nur in der Schweiz.
