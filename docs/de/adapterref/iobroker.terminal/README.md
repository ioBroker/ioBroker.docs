# IoBroker.terminal
Basierend auf [Web-Terminal](https://github.com/rabchev/web-terminal) von rabchev.

Terminalserver zum Öffnen der Befehlszeilenschnittstelle.
Bitte verwenden Sie es nur für Verwaltungszwecke.
