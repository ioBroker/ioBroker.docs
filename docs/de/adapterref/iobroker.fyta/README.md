## FYTA-Adapter für ioBroker
Ein inoffizieller Adapter zur Verbindung von ioBroker mit [FYTA Pflanzensensoren](https://fyta.de/) Sensoren kann auf der [FYTA Shop](https://fyta.de/collections/all) erworben werden.
