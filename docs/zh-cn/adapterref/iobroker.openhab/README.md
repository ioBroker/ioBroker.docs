＃ioBroker.openhab =================
此适配器将ioBroker与[openhab](http://openhab.org/)连接。

它从openhab导出设备和组，然后监视变量的更新。

＃＃ 去做
 - 新设备检测到同义而不重启适配器？
 - 目前只有项目查看/可切换，也可以阅读“事物”
 - 从OpenHab中删除的项目不会在ioBroker中消失
