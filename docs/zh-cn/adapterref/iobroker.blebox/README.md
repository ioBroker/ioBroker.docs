# IoBroker.blebox
## IoBroker 的 blebox 适配器
适配器 für Bleboxsysteme。详细信息 zu den Produkten [盲盒](https://blebox.eu/)。 Die Implementierung hat ohne Unterstützung des Herstellers stattgefunden。

Eine unvollständige und veraltete API-Simulation kann [希尔](https://github.com/blebox/blebox-virtual-devices) heruntergeladen werden。

Aktuell werden folgende Geräte unterstützt:

* 百叶箱
* 开关盒
* 桑拿箱
* 温度传感器
