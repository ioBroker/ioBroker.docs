# IoBroker.luftdaten
＃＃ 配置
＃＃＃ 当地的
1. 构建您自己的传感器并将其添加到您的本地网络
2.创建适配器的新实例
3. 在第一列输入自定义名称
4. 选择“本地”作为类型（第二列）
5.在第三栏填写传感器的IP地址或主机名
6.保存设置

等待几秒钟，直到 cronjob 第一次收集数据。

*随意更改实例选项卡中的计划设置（默认：每 30 分钟）。*

＃＃＃ 偏僻的
1.在官方地图上选择一个传感器：[sensor.community](https://sensor.community/en/)
2.点击传感器并复制ID（#XXXXX）
3.创建适配器的新实例
4. 在第一列输入自定义名称
5.选择“远程”作为类型（第二列）
6.在第三栏填写传感器的ID（不带#）
7.保存设置

等待几秒钟，直到 cronjob 第一次收集数据。

*随意更改实例选项卡中的计划设置（默认：每 30 分钟）。*
