＃ioBroker.g-homa
================

适用于G-Homa Wifi插头的ioBroker适配器。提供本地命令服务器，以便插头不再打电话回家。
现在必须使用该应用程序包含插件，但很快将在支持WiFi的设备上通过ioBroker支持。
