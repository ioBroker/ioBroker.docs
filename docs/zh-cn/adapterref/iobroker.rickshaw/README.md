使用[黄包车](https://shutterstock.github.io/rickshaw/examples/)绘制图表。

＃＃ 如何使用
 - 安装“socketio”-adapter并创建一个实例
 - 安装“web”-adapter并创建一个实例。
 - 在“web”设置中选择-adapter安装“socket.io”-instance的实例。
 - 安装“人力车”-adapter并致电
 - 转到http：// ip：8082 / rickshaw / edit.html
