## IoBroker 的cleveron 适配器
Erhalte deine Daten aus der CLEVERON - API (<https://www.cleveron.ch>)

＃＃ 手动的
- 你只需要你的cleveron-login-data。
- 适配器获取所有建筑物、房间和设备 - 由cleveron API 提供的数据。

- 添加所需的轮询 - 以分钟为单位的间隔。

- 如果您添加了新设备、房间或家庭，或者您更改了任何设置，请重新启动适配器。
