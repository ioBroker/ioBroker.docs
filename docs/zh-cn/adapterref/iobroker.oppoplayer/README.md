#ioBroker.oppoplayer
该适配器增加了对opo数字UHD播放器（UDP-20x）到ioBroker的支持。
您可以控制播放和查询状态。

＃＃ 提示与技巧
*如果再次连接电源（由oppo确认），播放器不会启动网络接口。

  如果断开他与电源的连接，则必须使用触发器启动它。
