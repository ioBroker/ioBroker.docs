# IoBroker.fiat
**测试：**![测试与发布](https://github.com/TA2k/ioBroker.fiat/workflows/Test%20and%20Release/badge.svg)

## 适用于 ioBroker 的菲亚特吉普适配器
适用于 Fiat 和 Jeep 的适配器

## 登录信息：
输入菲亚特或吉普应用程序的邮件和密码。

## 施托伊恩
fiat.0.id.remote 设置为 true 表示已锁定
