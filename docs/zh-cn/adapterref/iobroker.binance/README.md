＃ioBroker.binance
＃＃ 介绍
与加密货币交易平台币安进行通信的适配器

适配器在配置的更新间隔中拉动货币价格。
如果您配置了API密钥和相应的机密，它也会拉动帐户余额。
您可以在binance.com上创建一个API密钥-我建议将其限制为“只读”。
