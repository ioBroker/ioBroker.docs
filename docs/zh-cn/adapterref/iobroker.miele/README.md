## IoBroker.miele
####说明
适用于Miele设备的适配器

####配置
配置没有必要。第一次接收的广播将创建所有可用的设备和状态

####安装
在iobroker根目录中执行以下命令（例如在/ opt / iobroker中）

```
npm install iobroker.miele
```

＃＃＃＃ 要求
Miele网关，例如XGW 3000 <！ -
