＃ioBroker.hp-ilo =================
#i HPBO适用于ioBroker的适配器
将具有Integrated Lights-out（ILO）的Hewlett-Packard服务器的温度数据读入[ioBroker](https://www.iobroker.net)。

从V2向上支持ILO接口。
