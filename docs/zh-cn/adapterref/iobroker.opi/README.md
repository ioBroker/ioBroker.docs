＃ioBroker.opi ===================
用于集成到ioBroker的OPI-Monitor实现。

###重要信息
测试硬件：OrangePi plus2 H3

###选择后可以使用以下对象：
＃＃ *中央处理器*
 -  cpu_frequency
 -  load1
 -  load5
 -  load15

## *内存*
 -  memory_available
 -  memory_free
 -  memory_total

## *网络（eth0）*
 -  net_received
 -  net_send

## *eMMC*
 -  emmc_root_total
 -  emmc_root_used

## *交换*
 -  swap_total
 -  swap_used

## *温度*
 -  soc_temp

## *正常运行时间*
 - 正常运行时间

## *WLAN*
 -  wifi_received
 -  wifi_send

###配置
在配置页面上，您可以选择以下模块：

- 中央处理器
 - 记忆
 - 网络
 -  eMMC
 - 交换
 - 温度
 - 正常运行时间
 -  WLAN
