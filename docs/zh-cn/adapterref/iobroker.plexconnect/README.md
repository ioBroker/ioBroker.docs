#ioBroker.plexconnect
此适配器允许在第一步从Plex Media Server获取Webhooks。
随后将有更多功能。

###！注意！
*对于某些功能，您需要PlexPass。
*您必须在plex-server的网络设置中启用“Webhoks”才能获取消息！
* [关于Web-Hooks的更多信息请点击这里]（https://support.plex.tv/hc/en-us/articles/115002267687-Webhooks）
