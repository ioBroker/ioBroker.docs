# Официальное приложение ioBroker
Проект ioBroker имеет официальные приложения для Android и iOS. Приложения доступны в соответствующих магазинах приложений:

- [Приложение для Android](https://play.google.com/store/apps/details?id=com.iobroker.visu&hl=en_US)
- [iOS-приложение](https://apps.apple.com/de/app/iobroker-visu/id1673095774)

Использование приложения для локального доступа бесплатно. Для удаленного доступа (вне локальной сети) вам необходимо иметь подписку на облако [ioBroker.pro](https://iobroker.pro/www/pricing#remote).

## Функции
Приложение позволяет вам:

- Получите доступ к своему серверу ioBroker из локальной сети (бесплатно) или из любой точки мира (требуется подписка на облако)
- Управляйте устройствами умного дома с помощью vis-2, iQontrol или Lovelace
- Отправлять уведомления на ваш телефон
- Отправлять статус телефона в ioBroker, например уровень заряда батареи, местоположение и т. д.
