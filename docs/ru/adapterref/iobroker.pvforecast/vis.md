# IoBroker.pvforecast - ВИС
Существует два способа представления данных прогноза в ВИС.

- Таблица JSON (работает напрямую)
- JSON Graph / Chart (входит в состав ``vis-materialdesign``)

## JSON-таблица
Выберите ``JSONTable`` Object-ID als Datenquelle. Beispielsweise ``pvforecast.0.summary.JSONTable``.

## График/диаграмма JSON
Для использования виджета требуется адаптер [ioBroker.vis-материалдизайн](https://github.com/Scrounger/ioBroker.vis-materialdesign).

```
iobroker add vis-materialdesign
```

Выберите ``JSONGraph`` Object-ID als Datenquelle. Beispielsweise ``pvforecast.0.summary.JSONGraph``.

Некоторые свойства графика можно указать в настройках адаптера (например, поворот и размер шрифта надписей).
