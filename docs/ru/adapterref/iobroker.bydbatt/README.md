=================

byd Battery получить информацию от BYD Battery Box (солнечные батареи) ---------------------------------------------------- -----------------------------------------

<b>NodeJs &gt;&gt; 10.xx</b>

требуется пароль. вы можете установить его в настройках BYD Manager
