# IoBroker.letrika_comgw
## Letrika_comgw адаптер для ioBroker
Собирает данные с Letrika SMI 260 через коммуникационный шлюз letrika.

Адаптеру требуется как минимум nodejs версии 8 / ES9 / ECMAScript2018.
