# IoBroker.emporia
Этот адаптер получает данные из системы emporia Engergy. На данный момент только получение

- живое энергопотребление
- ежедневные значения

на данный момент нет розеток и солнечной батареи.

Использована документация по API из [здесь](https://github.com/magico13/PyEmVue/blob/master/api_docs.md).
