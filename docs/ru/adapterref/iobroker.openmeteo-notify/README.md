# IoBroker.openmeteo-notify
**Тесты:** ![Тестирование и выпуск](https://github.com/ipod86/ioBroker.openmeteo-notify/workflows/Test%20and%20Release/badge.svg)

## Адаптер ioBroker для прогнозов погоды Open-Meteo
Этот адаптер получает данные прогноза погоды из бесплатного источника [API Open-Meteo](https://open-meteo.com) и предоставляет их в виде точек данных ioBroker. Ключ API не требуется.

## Основные моменты
### Настраиваемый HTML-виджет
Адаптер генерирует готовый к использованию HTML-код (`widget`), который можно встроить непосредственно в VIS, vis-2 или любую панель управления ioBroker — без использования внешних инструментов или ручного CSS. Тема оформления (светлая/темная), прозрачность фона, прозрачность карточки, размер шрифта и цвет карточки настраиваются непосредственно в параметрах адаптера.

### Поиск по полному тексту адреса
Местоположение не обязательно вводить в виде прямых координат. В пользовательском интерфейсе настроек доступен **поиск адреса по свободному тексту** — просто введите город, адрес или регион, и координаты будут определены автоматически. Для каждого местоположения отображается предварительный просмотр в OpenStreetMap. Можно настроить несколько местоположений одновременно.

### Прогноз на срок до 16 дней
В зависимости от выбранной метеорологической модели, доступны ежедневные прогнозы на срок до **16 дней** — значительно больше, чем типичные 5–7 дней у большинства адаптеров.

### Тексты сводок погоды
Адаптер генерирует сводки погоды на естественном языке (`current.summary`, `dayX.summary_day`, `dayX.summary_night`) на **11 языках**, используя пороговые значения DWD для температуры, ветра и осадков, включая риск гроз на основе CAPE.

## Функции
— **Бесплатно и без ключа API** — Open-Meteo — это бесплатный API для прогноза погоды с открытым исходным кодом.
- **Несколько местоположений** – настройте столько местоположений, сколько вам нужно, для каждого из которых будет доступен поиск адреса и предварительный просмотр карты.
- **Резервный вариант определения местоположения системы** – использует координаты системы ioBroker, если местоположение не указано.
- **Настраиваемый диапазон прогноза** – до 16 дней в сутки, до 16 дней в час
- **Настраиваемый интервал обновления** – 60 мин, 120 мин или ежедневно в 01:00.
- **Единицы измерения** – температура (°C / °F), скорость ветра (км/ч, м/с, миль/ч, узлов), количество осадков (мм / дюйм)
- **5 наборов иконок погоды** с предварительным просмотром в реальном времени в настройках:
- Метеоконс от Баса Милиуса – статический PNG (по умолчанию)
- Метеоконс от Баса Милиуса – анимированный SVG
- Анимированные иконки погоды amCharts в формате SVG *(дождь/снег/гром: нет варианта день/ночь)*
- Иконки погоды amCharts – статический SVG *(дождь/снег/гром: нет варианта день/ночь)*
- Метеорологические символы ВМО OGC – Папуа-Новая Гвинея
- **Значки дня/ночи** – Значки облачности и ясности на Meteocons и amCharts переключаются на ночной вариант в зависимости от значения `is_day`.
- **Направление ветра** – градусы, текст компаса (С/СВ/В/…), эмодзи стрелки (⬆️↗️…), иконка стрелки в формате SVG.
- **Сила ветра** – шкала Бофорта (0–12) с иконками Бофорта от Meteocons.
- **`info.lastUpdate`** – метка времени последнего успешного обновления

### Дополнительные группы данных (переключаемые индивидуально, каждая с опцией "также почасовая")
| Группа | По умолчанию | Точки данных |
|-------|---------|-------------|
| **Качество воздуха** | on | european_aqi, PM10, PM2.5, NO₂, CO, dust, ozone → `current.air_quality` / `hXX.air_quality` |
| **Сельское хозяйство / Солнечная энергия** | выключено | солнечное излучение, CAPE, температура почвы, освещенность → `*.agriculture` |
| **Индексы комфорта** | выключено | индекс жары, индекс ветрового охлаждения, индекс влажности, УФ-индекс → `*.comfort` |
| **Пыльца** | выключено | ольха, береза, трава, полынь, олива, амброзия с уровнем текста → `dayX.pollen` / `hXX.pollen` |
| **Предупреждения DWD** | выключен | Официальные предупреждения Deutscher Wetterdienst (только для Германии) → `location.warnings.*` |
| **Предупреждения DWD** | выключен | Официальные предупреждения Deutscher Wetterdienst (только для Германии) → `location.warnings.*` |

Когда группа отключена, каналы с данными, относящиеся к ней, автоматически удаляются при следующем обновлении.

### Официальные предупреждения о погоде
Адаптер интегрирует официальные предупреждения о погоде от национальных метеорологических служб. Включите их с помощью переключателя «Официальные предупреждения о погоде». Служба выбирается автоматически на основе координат местоположения:

| Страна | Сервис | Зона покрытия |
|---------|---------|----------|
| Германия (DE) | [ДВД](https://www.dwd.de) – Deutscher Wetterdienst | Все типы предупреждений, 4 уровня серьезности |
| Страны ЕС | [MeteoAlarm](https://www.meteoalarm.org) | Все типы предупреждений, сопоставление на основе полигонов |
| Другое | — | Недоступно (используйте рассчитанные предупреждения Open-Meteo) |

Предупреждения хранятся в разделе `location.warnings.*` независимо от источника. Точка данных `warnings.source` соответствует `"DWD"` или `"MeteoAlarm"`.

## Установка
Установите через административный интерфейс ioBroker (найдите "openmeteo-notify").

## Точки данных
Адаптер создает точки данных в разделе `openmeteo-notify.<instance>.<location>`.

### Текущая погода (`current`)
| Точка данных | Описание | Единица измерения |
|-----------|-------------|------|
| `temperature` | Текущая температура | °C/°F |
| `weathercode` | Метеорологический код ВМО (0 = ясное небо, 95/99 = гроза) – полная таблица: [ВМО 4677](https://open-meteo.com/en/docs#weathercode) | |
| `description` | Удобочитаемое описание погоды (11 языков) | |
| `icon` | Эмодзи погоды | |
| `icon_url` | URL значка погоды (набор значков можно выбрать в настройках) | |
| `precipitation` | Общее количество осадков за последний час | мм/дюйм |
| `rain` | Количество осадков за последний час | мм/дюйм |
| `snowfall` | Снегопад за последний час | см |
| `snow_depth` | Текущая глубина снега на земле | см |
| `cloudcover` | Облачность | % |
| `humidity` | Относительная влажность | % |
| `dew_point` | Точка росы – температура, при которой воздух становится насыщенным; близкая к температуре воздуха = высокая влажность | °C/°F |
| `pressure` | Атмосферное давление, приведенное к среднему уровню моря (MSL) | гПа |
| `visibility` | Горизонтальная видимость | м |
| `is_day` | `true` между восходом и закатом солнца | логическое значение |
| `windspeed` | Скорость ветра (единица измерения на выбор: км/ч, м/с, миль/ч, узлов) | км/ч … |
| `windgusts` | Максимальная скорость порывов ветра | км/ч … |
| `winddirection` | Направление ветра (метеорологическое: направление, откуда дует ветер) | ° |
| `winddirection_text` | Текст, обозначающий стороны света | С/СВ/… |
| `winddirection_icon` | Эмодзи основных направлений | ⬆️↗️… |
| `winddirection_icon_url` | URL значка стрелки направления ветра | |
| `windbeaufort` | Сила ветра по шкале Бофорта (0 = штиль, 8 = штормовой ветер, 12 = ураган) | 0–12 |
| `windbeaufort_icon_url` | URL значка Бофорта | |
| `air_quality.*` | Индекс качества воздуха (AQI), PM10, PM2.5, NO₂, CO, пыль, озон *(если включено)* | |
| `pollen.*` | Текущее количество пыльцы каждого типа *(если включено)* | Зерна/м³ |
| `agriculture.solar_radiation` | Коротковолновое солнечное излучение на уровне земли *(если включено)* | Вт/м² |
| `agriculture.cape` | CAPE – Конвективная доступная потенциальная энергия: энергия, доступная для развития грозы; > 500 Дж/кг = значительный риск, > 2000 Дж/кг = высокий риск *(если включено)* | Дж/кг |
| `agriculture.soil_temp` | Температура почвы на глубине 0 см *(если включено)* | °C/°F |
| `comfort.heat_index` | Индекс жары (Ротфус) – насколько жарко ощущается температура и влажность; значимыми считаются только значения ≥ 27 °C и ≥ 40 % относительной влажности, `null` в противном случае *(если включено)* | °C/°F |
| `comfort.windchill` | Ощущаемая температура из-за ветра (NWS) – насколько холодно ощущается из-за ветра; имеет значение только при температуре ≤ 10 °C и скорости ветра > 4,8 км/ч, `null` в противном случае *(если включено)* | °C/°F |
| `comfort.humidex` | Индекс влажности (канадская формула) – комбинированный индекс дискомфорта от жары и влажности; значения выше 40 вызывают дискомфорт, выше 46 – опасность (если включено) | °C/°F |
| `comfort.humidex_level` | Уровень дискомфорта по шкале Humidex: 1 = отсутствует (<29) · 2 = незначительный (29–34) · 3 = заметный (35–39) · 4 = сильный (40–45) · 5 = опасный (≥46) *(если включено)* | 1–5 |
| `comfort.uv_index` | УФ-индекс (0–11+) – интенсивность УФ-излучения на уровне земли *(если включено)* | |
| `comfort.uv_level` | Уровень защиты от УФ-излучения (шкала ВОЗ): `low` (0–2, отсутствие защиты) · `moderate` (3–5, солнцезащитный крем) · `high` (6–7) · `very_high` (8–10) · `extreme` (≥11) *(если включено)* | |
| `comfort.uv_level` | Уровень защиты от УФ-излучения (шкала ВОЗ): `низкий` (0–2, отсутствие защиты) · `умеренный` (3–5, солнцезащитный крем) · `высокий` (6–7) · `очень высокий` (8–10) · `экстремальный` (≥11) *(если включено)* | |

## Отказ от ответственности
Этот адаптер использует данные из следующих сторонних сервисов:

- **[Open-Meteo](https://open-meteo.com)** – данные прогноза погоды. Название и логотип Open-Meteo являются собственностью соответствующих владельцев.
- **[DWD](https://www.dwd.de)** (Deutscher Wetterdienst) – официальные предупреждения о погоде для Германии. Название и данные DWD являются собственностью Deutscher Wetterdienst.
- **[MeteoAlarm](https://www.meteoalarm.org)** – официальные метеорологические предупреждения для европейских стран. Название и данные MeteoAlarm являются собственностью соответствующих владельцев.

Этот адаптер является независимым проектом сообщества и не связан ни с одним из вышеперечисленных сервисов и не поддерживается ими.
