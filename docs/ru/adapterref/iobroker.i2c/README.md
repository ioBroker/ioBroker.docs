# IoBroker адаптер для I2C
Связь с устройствами, подключенными к локальной системе, по шине I2C.

Этот адаптер должен работать на платах Linux, таких как Raspberry Pi, C.H.I.P., BeagleBone или Intel Edison.

## Установить
Перед установкой ознакомьтесь с [руководство по установке модуля i2c-bus](https://www.npmjs.com/package/i2c-bus#installation).

Особенно убедитесь, что вы правильно настроили и включили I2C в вашей системе (при необходимости):

* [Настройка I2C на Raspberry Pi] (https://github.com/fivdi/i2c-bus/blob/master/doc/raspberry-pi-i2c.md)
* [Настройка I2C на базовой плате Intel Edison Arduino] (https://github.com/fivdi/i2c-bus/blob/master/doc/edison-adruino-base-board-i2c.md)

После того, как вы включили и настроили I2C, вы можете установить этот адаптер через ioBroker Admin:

1. Запустите адаптер (он должен работать, чтобы обнаружение работало)
2. Откройте диалоговое окно конфигурации экземпляра.
3. Нажмите кнопку «Поиск устройств», чтобы обнаружить все подключенные устройства I2C - это займет некоторое время, наберитесь терпения!
4. Настройте все найденные устройства на соответствующих вкладках.
5. Сохраните конфигурацию (это перезапустит адаптер)

### Проблема прав доступа
В зависимости от возраста вашей установки ioBroker пользователь `iobroker` (или любой другой пользователь, на котором работает ioBroker) может не иметь надлежащих прав для доступа к I2C.

Если вы подключили устройство, но оно не отображается на экране конфигурации, убедитесь, что пользователь добавлен в группу `i2c`:

```sh
sudo usermod -G i2c iobroker
```

Примечание: если у вас нет стандартной установки, замените `iobroker` в приведенной выше команде на пользователя, выполняющего установку ioBroker (проверьте с помощью `ps`).

## Конфигурация
### Номер автобуса
Это номер открываемой шины / адаптера I2C, 0 для / dev / i2c-0, 1 для / dev / i2c-1, ...

На Raspberry Pi 3 это «1».

## Поддерживаемые устройства
В настоящее время поддерживаются следующие устройства. Числа в скобках - это известные адреса устройства в шестнадцатеричном формате (без считывающего бита).

### ADS1015 (48-4B)
Texas Instruments 4x 3.3-kSPS, 12-битные АЦП с внутренним заданием.

### ADS1115 (48-4B)
Texas Instruments 4x 860-SPS, 16-разрядные АЦП с внутренним заданием.

### BME280 (76, 77)
Bosch Цифровой датчик влажности, давления и температуры.

### MCP23008 8-битный расширитель ввода / вывода (20-27)
Микросхема 8-битного расширителя ввода / вывода с последовательным интерфейсом.

### MCP23017 16-битный расширитель ввода / вывода (20-27)
Микросхема 16-битный расширитель ввода / вывода с последовательным интерфейсом.

### PCF8574 8-битный расширитель ввода / вывода (20-27)
8-битный расширитель ввода / вывода Texas Instruments для шины I2C.

### PCF8574A 8-битный расширитель ввода / вывода (38-3F)
8-битный расширитель ввода / вывода Texas Instruments для шины I2C.
