# IoBroker.hp-ilo =================
# HP ILO адаптер для ioBroker
Считывает данные о температуре серверов Hewlett-Packard с интегрированным выходом (ILO) в [ioBroker](https://www.iobroker.net).

Поддержка интерфейсов ILO от V2 и выше.
