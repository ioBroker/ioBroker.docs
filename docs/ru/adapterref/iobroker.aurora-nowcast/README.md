## Адаптер Aurora Nowcast для ioBroker
Предоставляет **актуальные (прогнозируемые) данные** об активности северного и южного сияния для заданного местоположения на основе общедоступных данных Центра прогнозирования космической погоды NOAA (SWPC).

> **Примечание:** > Данные о северном сиянии OVATION представляют *текущие условия (прогноз на данный момент)*, основанные на измерениях солнечного ветра в реальном времени, а не долгосрочный прогноз.
> Кроме того, данные индекса Kp предоставляют **72-часовой прогноз** для целей планирования.

---

## Функции
- Получает данные об активности полярного сияния в режиме реального времени (модель NOAA OVATION) как для северного, так и для южного полушария.
- Вычисляет вероятность видимости северного сияния в заданном местоположении.
— Предоставляет текущий индекс Kp (с интервалом в 1 минуту) и прогноз Kp на 72 часа.
- Предоставляет данные о солнечном ветре в режиме реального времени (Bz, полное поле, скорость, плотность) в качестве индикаторов раннего предупреждения о полярных сияниях.
- Предоставляет состояния ioBroker для автоматизации, визуализации и оповещений.
- Возможность использования местоположения системы или ручного ввода широты/долготы.
— Подходит для панелей мониторинга, уведомлений и сценариев «умного дома»

## Конфигурация
Вы можете сделать одно из двух:

- Используйте местоположение системы, указанное в ioBroker, или
- Укажите координаты вручную (широта/долгота в десятичных градусах).

Если определение местоположения системой отключено, необходимо ввести координаты вручную.

Примеры:

| Местоположение | Широта | Долгота |
|-----------------|----------|-----------|
| Берлин | 52,5 | 13,4 |
| Буэнос-Айрес | -34,6 | -58,4 |
| Рейкьявик | 64.1 | -21.9 |

Значения для северо-восточного направления положительные, значения для юго-западного направления отрицательные.

## Штаты
### Справочная информация: Индексы космической погоды
**Индекс Kp** — Планетарный индекс K измеряет глобальную геомагнитную активность по шкале от 0 до 9 (0 = спокойная, 9 = сильная буря). Значения ≥ 5 указывают на условия геомагнитной бури (G1 и выше), при которых полярное сияние становится видимым в средних широтах, таких как Центральная Европа. Адаптер предоставляет как текущее значение Kp за 1 минуту, так и прогноз на 72 часа.

### ОВАЦИЯ — Вероятность северного сияния
| Штат | Тип | Описание |
|---------------------|---------|------------------------------------------------------------------------------------|
| `probability` | число | Предполагаемая вероятность видимости северного сияния в указанном месте (%) |
| `forecast_time` | число | Время, для которого рассчитывается прогнозируемый геомагнитный отклик Земли (UTC, мс) |
| `forecast_time` | число | Время, для которого рассчитывается прогнозируемый геомагнитный отклик Земли (UTC, мс) |

### Индекс Kp
| Штат | Тип | Описание |
|------------------------|---------|--------------------------------------------------------|
| `kp.value` | число | Текущий индекс Kp (0–9, десятичное значение, интервал 1 минута) |
| `kp.g_scale` | число | Производная шкала G NOAA (0 = нет, 1–5 = G1–G5) |
| `kp.forecast_max` | число | Максимальное значение Kp в 72-часовом прогнозе |
| `kp.forecast_max_time` | число | Время, когда достигается прогнозируемый максимум (UTC, мс) |
| `kp.forecast` | строка | Полный 72-часовой прогноз Kp в виде массива JSON `[{time, kp}]` |
| `kp.forecast` | строка | Полный 72-часовой прогноз Kp в виде массива JSON `[{time, kp}]` |

### Солнечный ветер
**Bz (GSM)** — z-компонента межпланетного магнитного поля в координатах GSM. Сильно отрицательное значение Bz (ориентация на юг) открывает магнитосферу Земли для энергии поступающего солнечного ветра и является наиболее надежным краткосрочным предвестником полярных сияний — обычно за 15–60 минут до видимой активности. **Bt** — это полная величина поля; Bz относительно Bt указывает, насколько сильно поле ориентировано на юг.

| Штат | Тип | Единица измерения | Описание |
|-------------------------|--------|--------|----------------------------------------------------------|
| `solar_wind.bz` | число | нТ | Компонент Bz в координатах GSM (отрицательное значение = на юг) |
| `solar_wind.speed` | число | км/с | Скорость протонов солнечного ветра |
| `solar_wind.density` | число | п/см³ | Плотность протонов солнечного ветра |
| `solar_wind.mag_time` | число | мс | Время измерения магнитометром (UTC) |
| `solar_wind.plasma_time`| число | мс | Время измерения плазмы (UTC) |
| `solar_wind.plasma_time`| число | мс | Время измерения плазмы (UTC) |

Эти состояния могут использоваться для:

- Уведомления (например, push-уведомления, когда Kp ≥ 5 или Bz ≤ −10 нТ)
- Визуализация данных на панели мониторинга
- Правила автоматизации (например, активировать камеру при высокой вероятности северного сияния)

## Практическое руководство: как увидеть северное сияние
Наблюдение за северным сиянием осуществляется в два этапа: **планирование заранее** с использованием прогноза Kp и **реагирование в режиме реального времени** с использованием данных о солнечном ветре и данных OVATION.

### Этап 1 — План: ожидается ли шторм?
Используйте `kp.forecast_max`, чтобы проверить, ожидается ли геомагнитная буря в ближайшие 72 часа. Примерные пороговые значения видимости в зависимости от географической широты:

| `kp.forecast_max` | Уровень шторма | Видно до ~ |
|-------------------|-------------|-------------------------------------------------|
| < 5 | Нет | Только высокие широты |
| 5 (G1) | Малый | ~60°N — северная Шотландия, южная Скандинавия |
| 6 (G2) | Умеренный | ~55°N — северная Германия, Польша |
| 7 (G3) | Сильный | ~50°N — Лондон, Франкфурт, Варшава |
| 8 (G4) | Сильный | ~45°N — Швейцария, Австрия, северная Италия |
| 9 (G5) | Экстремальный | ~40°N — центральная Франция, северная Испания |

`kp.forecast_max_time` показывает, *когда* ожидается пик активности — полезно для таких уведомлений, как «Прогноз шторма G2 на сегодня вечером».

`kp.g_scale` отражает текущий уровень шторма в реальном времени (0 = спокойно, 1–5 = G1–G5).

> **Примечание:** Это приблизительные географические широты Европы. Фактическая видимость сильно зависит от `solar_wind.bz` (см. ниже), облачности и светового загрязнения.

### Этап 2 — Реакция: активно ли сейчас северное сияние?
Даже при высоком значении Kp полярное сияние становится видимым только тогда, когда межпланетное магнитное поле (ММП) поворачивает на юг — на что указывает сильно отрицательное значение `solar_wind.bz`. Это наиболее надежный кратковременный триггер.

| `solar_wind.bz` | Значение |
|-----------------|----------------------------------------------------------|
| > 0 нТл | На севере — магнитосфера в значительной степени замкнута, полярное сияние слабое |
| От 0 до −5 нТл | Слабое смещение на юг — неблагоприятные условия |
| −5 до −10 нТл | На юге — начинается нарастание полярного сияния |
| ≤ −10 нТл | Сильно смещено на юг — вероятно значительное полярное сияние |
| ≤ −20 нТл | Экстремальное — интенсивное полярное сияние, простирающееся далеко в средние широты |

**Время опережения:** Bz измеряется в точке наблюдения L1 между Землей и Солнцем. Солнечному ветру требуется **15–60 минут**, чтобы пройти от L1 до Земли — это ваш предупреждающий период.

`solar_wind.bt` — это суммарная напряженность поля. Когда `|bz|` приближается к `bt`, поле почти полностью направлено на юг. Например, bz = −18 нТл при bt = 20 нТл — это более сильный сигнал, чем bz = −10 нТл при bt = 30 нТл.

`solar_wind.speed` усиливает эффект: сильный ветер (> 400 км/с) в сочетании с отрицательным Bz передает больше энергии в магнитосферу. Очень высокие скорости (> 600 км/с) могут вызывать полярное сияние даже при умеренном Bz.

`solar_wind.density` играет вспомогательную роль: высокая плотность (> 10 п/см³) увеличивает динамическое давление и может усиливать активность.

### Подтверждение, зависящее от местоположения: что добавляет `probability`?
Kp — это глобальный индекс, описывающий общую геомагнитную активность, а не то, что происходит над вашим местоположением. `probability` отличается: он рассчитывается специально для заданных вами координат с использованием **модели NOAA OVATION**, которая принимает в качестве входных данных измерения солнечного ветра в реальном времени и моделирует фактическую протяженность и интенсивность овального полярного сияния. Поэтому он реагирует быстрее и точнее на изменения Bz, чем производное значение Kp.

Для Центральной Европы (примерно 50–55° с.ш.) реалистичные диапазоны в условиях активной погоды:

| `probability` | Интерпретация |
|---------------|--------------------------------------------------------|
| < 5 % | Отсутствие значимой активности в вашем местоположении |
| 5–15 % | Повышенный уровень — стоит посмотреть, особенно вдали от городов |
| 15–30 % | Активное явление — северное сияние, вероятно, будет видно при ясном небе |
| > 30 % | Высокие накладные расходы |

Используйте `probability` в качестве подтверждения, специфичного для местоположения, в дополнение к Kp и Bz. Рост значения наряду с резко отрицательным значением Bz — это самый явный признак того, что стоит выйти на улицу.

### Пример логики автоматизации
Практическая трехуровневая стратегия оповещения:

1. **Режим наблюдения** — `kp.forecast_max` ≥ 5: «В ближайшие 72 часа ожидается шторм — следите за погодными условиями сегодня ночью»
2. **Предупреждение** — `kp.value` ≥ 5 И `solar_wind.bz` ≤ −10: «Активна гроза, Bz сильно смещается на юг — северное сияние вероятно через 15–60 минут»
3. **Подтверждение сверху** — «вероятность» ≥ 15: «Северное сияние, вероятно, видно в вашем местоположении прямо сейчас»

Объединение всех трех слоев позволяет избежать ложных срабатываний: фильтр Kp подтверждает наличие реальной бури, фильтр Bz подтверждает открытость магнитосферы, а фильтр вероятности подтверждает активность именно в вашем местоположении.

## Источник данных
Этот адаптер использует общедоступные данные, предоставленные:

- Центр прогнозирования космической погоды NOAA (SWPC)

<https://www.swpc.noaa.gov/>

В частности, для оценки активности полярных сияний в заданном местоположении используется модель прогнозирования полярных сияний OVATION и связанные с ней геомагнитные индексы в реальном времени.

---

## Отказ от ответственности
NOAA и SWPC не имеют отношения к этому проекту.

Данные, используемые этим адаптером, предоставлены NOAA для общественного пользования.
Не предоставляется никаких гарантий относительно точности, полноты или актуальности предоставленной информации.

Видимость северного сияния зависит от множества внешних факторов (например, облачности, светового загрязнения, ориентации межпланетного магнитного поля), которые выходят за рамки возможностей данного адаптера.
