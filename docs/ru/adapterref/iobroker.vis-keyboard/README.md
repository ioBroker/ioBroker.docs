# IoBroker.vis-клавиатура
![Скриншот](../../../en/adapterref/iobroker.vis-keyboard/img/widgets.png)

![Скриншот](../../../en/adapterref/iobroker.vis-keyboard/img/numpad.png)

Использован очень хороший плагин jquery от Джереми Саттерфилда. (Оригинал Джереми Саттерфилда, обновлен и поддерживается Робом Гаррисоном)

github — http://github.com/Mottie/Keyboard

## Использование
Вы можете разместить этот виджет на экране, и все виджеты с полями ввода будут иметь виртуальную клавиатуру. Виджет необходимо разместить на каждом экране.

Кроме того, вы можете добавить к классу виджета `keyboard-num` для принудительного использования цифровой клавиатуры или `keyboard-all` для принудительного использования полнотекстовой клавиатуры.
