## Important Information

tested Hardware: Asus Tinker Board


## Following Objects are available after selection:

### CPU
* cpu_frequency
* load1
* load5
* load15

### Memory
* memory_available
* memory_free
* memory_total

### Network (eth0)
* net_received
* net_send

### sdcard
* sdcard_root_total
* sdcard_root_used

### Swap
* swap_total
* swap_used

### Temperature
* soc_temp

### Uptime
* uptime

### WLAN
* wifi_received
* wifi_send

---

## Configuration
On configuration page you can select following modules:

* CPU
* Memory
* Network
* sdcard
* Swap
* Temperature
* Uptime
* WLAN
